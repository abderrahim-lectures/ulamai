\section{Ring Theory}

\begin{theorem}[domain\_is\_ufd]
Every integral domain $R$ is a unique factorization domain.
\end{theorem}

\begin{proof}
Every nonzero nonunit element of an integral domain factors into
irreducibles by a Noetherian induction argument: if some element admits no
factorization, a strictly ascending chain of principal ideals is
generated, which must eventually stabilize. Every irreducible element of
an integral domain is prime, since irreducibility already rules out
nontrivial factorizations of any element it divides. Hence the
factorization into irreducibles is unique up to order and associates.
\end{proof}
